ESTRUCTURA INTERNA

Cuando empezamos un documento de HTML, hay una estructura que es imprescindible poner. Y es la siguiente:

Pruébalo tú mismo


!DOCTYPE html: es la declaración que define este documento como HTML5.

html: el elemento raíz de una página HTML.

head: el elemento que contiene metainformación sobre el documento.

title: el elemento que especifica un título para el documento (aparece como nombre/título de pestaña/ventana).

body: el elemento que contiene el contenido visible de la página.

h1: el elemento que define un encabezado grande.

p: el elemento que define un párrafo.
