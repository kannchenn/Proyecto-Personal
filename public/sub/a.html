TABLAS

Para definir una tabla utilizamos la etiqueta table

Cada fila de tabla se define con la etiqueta tr. Antes de poner la fila debemos poner la etiqueta th. Finalmente, se define una tabla de datos/celda con la etiqueta td.

Pruebalo tu mismo

Para hacer más columnas en una celda, utilizamos el atributo colspan:

<th colspan="2">Texto</th>

Pruebalo tu mismo

Para hacer más filas en una celda, utilizamos el atributo rowspan:

<th rowspan="2">Texto</th>

Pruebalo tu mismo

Para agregar un título a la tabla, utilizamos la etiqueta caption:

<caption>Título</caption>
