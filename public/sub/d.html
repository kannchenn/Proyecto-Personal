CLASES

Class es un atributo. Normalmente se añade a elementos de tipo block e inline para facilitar a la hora de codificar. Lo utilizaremos cuando aprendamos CSS. Pero para emepzar, debemos saber un poco sobre él y para qué sirve, en concreto.


Cuando queremos escribir en tres secciones sobre textos diferentes, pero queremos que sigan el mismo estilo, podemos utilizar el atributo class para no repetir tantas veces las propiedades de estilo. El estilo concreto se programará dentro del CSS.

<p class="class"> Se escribe así. </p>
