IFRAMES

Un iframe se utiliza para mostrar una página web dentro de una página web.

Y la sintaxi básica es:

<iframe src="URL"></iframe>

URL=link de la página web que se quiere enlazar.

Dicho eso, podemos dar un poco de estilo a este iframe.

  • Quitar el border

    • <iframe src="https://es.wikipedia.org/wiki/Iframe" style="border:none;"></iframe>

  • Cambiar el estilo, tamaño y color

    • <iframe src="https://es.wikipedia.org/wiki/Iframe" style="border:2px solid grey;"></iframe>
