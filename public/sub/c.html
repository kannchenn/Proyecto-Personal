BLOQUES

Un elemento que es tipo bloque siempre comienza en una nueva línea y ocupa todo el ancho disponible.

El <div> element es así.

Cuando hablamos de bloques, no referimos a etiquetas como por ejemplo:

  • <div>
  • <p>
  • <form>
  • <h1> - <h6>

DIV

El elemento div se utiliza a menudo como un contenedor para otros elementos HTML. No tiene atributos obligatorios, aunque si utilizamos los atributos style y class. 

 <div style="background-color:black;color:white;padding:20px;">
  <h2>Londres</h2>
  <p>Me gustaria visitar Londres.</p>
</div>

INLINE

Un elemento que es tipo inline no comienza en una nueva línea y solo ocupa lo que se requiere el texto.

Esto es un inline <span> element.

Cuando hablamos de bloques, no referimos a etiquetas como por ejemplo:

  • <span>
  • <a>
  • <img>

SPAN

El elemento span es lo mismo que el elemento div pero recuerda que es inline element! No tiene atributos obligatorios, aunque si utilizamos los atributos style y class. 

 <h1>Esto<span style="color:grey">es</span>una web</h1>
