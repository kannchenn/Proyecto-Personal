Alineación de texto

Utilizamos la propiedad text-align para definir la alineación horizontal del texto para un elemento HTML. El valor puede ser:

  • left = Las líneas de texto se representan a la izquierda
  • right = Las líneas de texto se representan a la derecha
  • center = Las líneas de texto están centradas
  • justify = Las líneas de texto están justificadas para ambos márgenes
