LINK

Puedes hacer clic en un enlace y saltar a otro documento. Cuando mueves el ratón sobre un enlace, la flecha del ratón se convertirá en una pequeña mano. Además, no hace falta que sea el link en sí, sino que puede ser un texto aparte hasta una imagen.

Utilizamos la etiqueta a para definir a un link. El href es atibuto que especificará el valor/estinación del link. link text es la parte visible.

El atributo target

El atributo target especifica dónde abrir el documento vinculado. Puede tener diferentes valores:

  • _blank: Abre el documento vinculado en una nueva pestaña.
  • _self: Abre el documento vinculado en la misma ventana / pestaña que se hizo clic (es el predeterminado).
  • _parent: Abre el documento vinculado en el marco padre.
  • _top: Abre el documento vinculado en todo el cuerpo de la ventana.
  • framename: Abre el documento vinculado en un marco con nombre.

Marcador en la misma página.

A veces la página web puede estar muy larga, entonces utilizamos los marcadores para permitir a los lectores a saltar a partes específicas de una página Web.

  1. Primero, debemos de utilizar el atributo id para crear el marcador.

    2. <h2 id="name">Aprender a programar</h2>

  2. Después, debemos de agregar un enlace al marcador en tu índice o cualquier otro texto, desde la misma página, utilizando el atributo href con un # delante del valor. El nombre del valor debe ser el mismo que del atributo id.

    3. <a href="#name">Visita a Aprender a programar</a>

  3. Así, si clicas en el texto del punto 2, entonces se dirigirá al punto 1. Recuerda que el nombre del valor puede variar, pero siempre debe coincidir entre el atributo id y href que se quiere enlazar.

Link Locales y Externas

Para vincular a una página web externa de la red, ponemos el url completo en el valor:

<a href="http://www.google.es">Google</a>

Sin embargo, si queremos enlazar con una página ubicada en el local, deberíamos utilizar la siguiente manera. Para entenderlo mejor, utilizaremos una imagen para explicar.

Para enlazar desde index.html hacia yyy.html:

<a href="seccion1/paginas/yyy.html">ir a yyy.html</a>

Para hacer un enlace desde xxx.html hacia yyy.html:

<a href="../seccion1/paginas/yyy.html">Ir (también) a yyy.html</a>

Para hacer un enlace desde yyy.html hacia xxx.html:

<a href="../../seccion2/xxx.html">Ir ahora a xxx.html</a>
