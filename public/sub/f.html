HEAD

El elemento head es un contenedor de metadatos (datos sobre datos) y se coloca entre la etiqueta html y la etiqueta body. La metadata no son mostradas en las páginas webs, sirven para demostrar informaciones(el título del documento, el conjunto de caracteres, los estilos, los vínculos, los scripts,etc.) sobre el documento HTML. Las etiquetas que se utilizan dentro de head son:

  • <title>

    • Este elemento defina el título en la pestaña del navegador del documento HTML, proporciona un título para la página cuando se añade a los favoritos, muestra un título para la página en los resultados de los motores de búsqueda.

  • <style>

    • Este elemento sirve para definir información del estilo para una página HTML (CSS)

  • <meta>

    • Se utiliza para especificar qué conjunto de caracteres queremos utilizar, la descripción de la página, las palabras clave, el autor y otros metadatos. Por ejemplo:

    • Para definir el conjunto de caracteres utilizado:

      • <meta charset="UTF-8">

    • Para definir una descripción de la página web:

      • <meta name="description" content="La Programación">

    • Para definir palabras clave para los motores de búsqueda:

      • <meta name="keywords" content="HTML, CSS">

    • Para definir el autor de la página:

      • <meta name="author" content="Kangxin Chen">

    • Para actualizar documento cada 30 segundos:

      • <meta http-equiv="refresh" content="30">

  • <link>

    • Este elemento se utiliza para vincular a hojas externas. (para vincular con CSS, Javascript...) Por ejemplo:

    <link rel="stylesheet" href="mystyle.css">

  • <script>

    • Este elemento se utiliza para escribir códigos de Javascript y dar esta información a la página web. Por ejemplo:

    <script>
function myFunction {
    document.getElementById("demo").innerHTML = "Hello JavaScript!";
}
</script>
  • <base>

    • Especifica la URL base y el objetivo base para todas las URL relativas de una página. Por ejemplo, si queremos que se abra el link en una nueva página:

    <base href="http://www.w3schools.com/images/" target="_blank">
