LISTAS

Hay tres tipos de listas:

Lista no ordenada

Una lista desordenada comienza con la etiqueta ul. Cada elemento de la lista empieza con la etiqueta li. Se marcan con viñetas de forma predeterminada:

 <ul>
  <li>Café</li>
  <li>CocaCola</li>
  <li>Agua</li>
</ul>

También puedes cambiar la viñeta por un otro tipo de signo. 

 <ul style="list-style-type:disc">
 <ul style="list-style-type:circle">
 <ul style="list-style-type:square">
 <ul style="list-style-type:none">

Pruébalo tú mismo

Lista ordenada

Una lista ordenada comienza con la etiqueta ol. Cada elemento de la lista empieza con la etiqueta li. Se marcan con números de forma predeterminada:

 <ol>
  <li>Café</li>
  <li>CocaCola</li>
  <li>Agua</li>
</ol>

También puedes cambiar los números por un otro tipo de ordenación. 

 <ul type="1">
 <ul type="A">
 <ul type="a">
 <ul type="I">
 <ul type="i">

Pruébalo tú mismo

Lista descriptiva

Una lista de descripción es una lista de términos, con una descripción de cada término. La etiqueta dl define la lista de descripción, la etiqueta dt define el término (nombre) y la etiqueta dd describe cada término:

 <dl>
  <dt>Café</dt>
  <dd>- con galletas</dd>
  <dt>Leche</dt>
  <dd>- con chocolate</dd>
</dl>
