IMÁGENES

Se definen con la etiqueta img. La etiqueta img no puede ir solo, contiene atributos solamente y no tiene una etiqueta de cierre. El atributo src especifica la URL (dirección web) de la imagen:

<img src="url" alt="some_text" style="width:width;height:height;">

Atributo alt

El atributo alt proporciona un texto alternativo para una imagen, si por alguna razón el usuario no puede verla (debido a una conexión lenta, un error en el atributo src o si el usuario usa un lector de pantalla), mostrará el valor del atributo alt:

<img src="nombre_equicocado.gif" alt="Google Icon";">

Tamaño de imagen

Para editar la foto, se puede utilizar el atributo style para especificar por ejemplo la anchura y altura. La anchura y altura se especifica con pixeles:

<img src="html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

O directamente utilizando el atributo width y height:

<img src="html5.gif" alt="HTML5 Icon" width="128" height="128";">

Posición de imagen

Para mover la imagen a la derecha o a la izquierda de un texto, utilizamos la propiedad float.:

<img src="smiley.gif" alt="Smiley face" style="float:right;width:42px;height:42px;">

<img src="smiley.gif" alt="Smiley face" style="float:left;width:42px;height:42px;>

Mapa de imágen

La etiqueta map sirve para definir un mapa de imágenes. Un mapa de imágenes es una imagen con áreas que se pueden hacer clic.

El atributo name de la etiqueta map está asociado al atributo usemap del img y crea una relación entre la imagen y el mapa.
