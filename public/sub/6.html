CÓDIGO DE COMPUTADORA

Cuando queremos mostrar el código de computadora, utilizamos los siguientes elementos para muestrar en tamaño de letra fija y espaciado:

kbd

para definir la entrada de teclado

<kbd>File | Open...</kbd>

Resultado:

File | Open...

Pruebalo tu mismo

samp

para definir la salida de muestra de un programa de computadora

<samp>demo.example.com login: Apr 12 09:10:17 Linux 2.6.10-grsec+gg3+e+fhs6b+nfs+gr0501+++p3+c4a+gr2b-reslog-v6.189 </samp>

Resultado:

demo.example.com login: Apr 12 09:10:17 Linux 2.6.10-grsec+gg3+e+fhs6b+nfs+gr0501+++p3+c4a+gr2b-reslog-v6.189

Pruebalo tu mismo

code

para definir una pieza de código de programación

<code> var x = 5; var y = 6; document.getElementById("demo").innerHTML = x + y; </code>

Resultado:

var x = 5; var y = 6; document.getElementById("demo").innerHTML = x + y;
