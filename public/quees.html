INTRODUCCIÓN

La programación informática de computadores es el proceso de diseñar, codificar, depurar y mantener el código fuente de programas computacionales. El código fuente es escrito en diferentes lenguajes de programación. Es decir, el ordenador es una máquina con extraordinarias capacidades para realizar muchas cosas con facilidad (matemáticas, gráficos,etc) pero incapaz de hacer nada por sí mismo y necesita que un humano lo 'programe'. Y ese humano está realizando la programación.

Por lo tanto, diríamos que programar es definir instrucciones para ser ejecutadas por un ordenador. El resultado de esta tarea es un paquete de instrucciones al que llamamos programa. El objetivo de programar suele ser resolver un problema.


LENGUAJES DE PROGRAMACIÓN

Para definir instrucciones a un ordenador tenemos que utilizar un lenguaje o código específico. Los lenguajes de programación, salvando las distancias, tienen similitudes con las lenguas humanas: evolucionan. Por eso cada día puede que se esté aumentando más números de vocablos o directamente nuevos lenguajes. Se expanden internacionalmente y triunfan si gran un número de personas decide a utilizarlos, o caen en desuso y se convierten en lenguas fósiles. Pueden ser más o menos fáciles de aprender, más o menos ricos en vocablos, más o menos fuertes para resistir el embate de las necesidades de los programadores. Por lo tanto, existe muchísimos códigos diferentes, aunque aquí solo vamos a aprender los más populares e fáciles de utilizar.

HTML: En principio HTML no es un lenguaje de programación, aunque de forma coloquial muchas veces se oigan referencias a HTML como si lo fuera. HTML(HyperText Markup Language) es un lenguaje de etiquetas. Estas etiquetas HTML comunican al navegador cuál es la información a mostrar por pantalla, además del formato de dicha información. Es por ello que no puede definirse como un lenguaje de programación, sino como un sistema de etiquetas. Veámoslo con un ejemplo.

CSS: CSS (Cascading Style Sheet) tampoco es un lenguaje de programación. Es un lenguaje para dotar la presentación y el estilo de un página web (documentos HTML). Podríamos decir que es un lenguaje que suele aparecer relacionado o próximo a un lenguaje de programación o que suele colaborar con un lenguaje de programación, pero no es un lenguaje de programación propiamente dicho.

JavaScript: JavaScript si que es un lenguaje de programación, se ejecuta con un ordenador, no en el servidor, permitiendo de crear efectos atractivos y dinámicos en las páginas web. Los navegadores modernos interpretan el código JavaScript integrado en las páginas web.

PHP: PHP (Hypertext Preprocessor) es un lenguaje de código abierto y adecuado para desarrollo web y que puede ser incrustado en HTML. Es popular porque un gran número de páginas y portales web están creadas con PHP. PHP se utiliza para generar páginas web dinámicas, es decir que cuyo contenido no es el mismo siempre. Por ejemplo, los contenidos pueden cambiar en base a los cambios que haya en una base de datos, de búsquedas o aportaciones de los usuarios, etc.


¿POR QUÉ PROGRAMAR?

Vivimos en una era donde la tecnología se ha convertido algo muy esencial para nosotros. Todo los que tenemos rodeado que sea tecnológico son casi todos programados, es decir, programación es todo. Me imagino que usted está aquí también para aprender esto y acercarse a este mundo. Atendiendo a que los ordenadores intervienen en la casi totalidad de las actividades que realiza el ser humano, y que la programación es lo que hace funcionar a los ordenadores, que programar sirve para todo.

Esta página web no incluye todo sobre la programación, sino que nos centraremos más en la creación de una página web. Y es por eso, anteriormente he listado lenguajes relacionados con la elaboración de paginas webs (html).
