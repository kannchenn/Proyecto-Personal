BOX MODEL CSS

Todos los elementos HTML se pueden considerar como cajas. En CSS, el término "modelo de caja" se usa cuando se habla de diseño. El modelo de caja CSS es esencialmente una caja que rodea cada elemento HTML. Consiste en: márgenes, bordes, relleno, y el contenido real. La imagen de abajo ilustra el modelo de caja:

  • Contenido - El contenido del cuadro, donde aparecen texto e imágenes.
  • Padding - Un área alrededor del contenido. El relleno es transparente.
  • Border - Una frontera que circunda el relleno y el contenido.
  • Margen - Un área fuera de la frontera. El margen es transparente.
div {
    width: 300px;
    border: 25px solid green;
    padding: 25px;
    margin: 25px;
}
