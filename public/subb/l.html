CSS LAYOUT - OVERFLOW

La propiedad de overflow CSS especifica si se debe recortar contenido o agregar barras de desplazamiento cuando el contenido de un elemento es demasiado grande para caber en un área especificada.:

  • visible: la forma predeterminada. El overflow es visible, lo que significa que no está recortado y se hace fuera de la caja del elemento: 
    div {
    width: 200px;
    height: 50px;
    background-color: #eee;
    overflow: visible;
}
    Puedes utilizar la propiedad de overflow cuando desea tener un mejor control del diseño. La propiedad de overflow especifica qué ocurre si el contenido desborda el cuadro de un elemento.

  • hidden: el overflow se recorta y el resto del contenido se oculta: 
    div {
    overflow: hidden;
}
    Puedes utilizar la propiedad de overflow cuando desea tener un mejor control del diseño. La propiedad de overflow especifica qué ocurre si el contenido desborda el cuadro de un elemento.

    Pruébalo tú mismo

  • scroll: el overflow se acorta y se agrega una barra de desplazamiento para desplazarse dentro del cuadro. Tenga en cuenta que esto agregará una barra de desplazamiento horizontal y verticalmente (incluso si no lo necesitas): 
    div {
    overflow: scroll;
}
    Puedes utilizar la propiedad de overflow cuando desea tener un mejor control del diseño. La propiedad de overflow especifica qué ocurre si el contenido desborda el cuadro de un elemento.

    Pruébalo tú mismo

  • auto: El valor automático es similar a scroll, sólo que añade barras de desplazamiento solo cuando sea necesario: 
    div {
    overflow: auto;
}
    Puedes utilizar la propiedad de overflow cuando desea tener un mejor control del diseño. La propiedad de overflow especifica qué ocurre si el contenido desborda el cuadro de un elemento.
