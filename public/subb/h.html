¿CÓMO INSERTAR CSS?

Hay 3 maneras de insertar CSS:

  1. External style sheet
  2. Internal style sheet
  3. Inline style

External style sheet

Puedes cambiar el aspecto de un sitio web entero con sólo un archivo externo.

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

Para vincular el documento css con el documento HTML, se debe escribir una elemento en la sección de head. Y dentro del head, se debe incluir una referencia al archivo de hoja de estilo externo dentro del elemento link .

La hoja CSS externa se puede escribir en cualquier editor de texto, pero no debe contener ninguna etiqueta html y se debe guardar con una extensión .css.

Internal style sheet

Utilizamos éste en el mismo documento HTML que queremos modificar su estilo, en la sección de head y lo definimos con la etiqueta style.

<head>
<style>
body {
    background-color: linen;
}

h1 {
    color: maroon;
    margin-left: 40px;
} 
</style>
</head>

Inline Styles

Se puede aplicar un estilo único en un solo elemento. Para utilizar estilos en línea, se debe añadir el atributo de estilo al elemento pertinente. El atributo de estilo puede contener cualquier propiedad CSS.

<h1 style="color:blue;margin-left:30px;">Esto es un heading.</h1>
